📰 RSS Reader

Connect Firebase to sync feeds, read state, and filter decisions across all your devices.

Paste the config object from Firebase console → Project settings → Your apps → Web app. Make sure it includes databaseURL (requires Realtime Database to be enabled).
Step-by-step setup instructions
  1. Go to console.firebase.google.com → Add project (disable Google Analytics if you don't need it).
  2. Add a Web App: click the </> icon → give it a nickname → Register app. Copy the firebaseConfig object that appears.
  3. Enable Authentication: Build → Authentication → Get started → Sign-in method → Google → Enable → Save.
  4. Enable Realtime Database: Build → Realtime Database → Create database → choose a region → Start in test mode (we'll tighten rules in step 5).
  5. Set security rules: in the Realtime Database console, click the Rules tab and paste:
    { "rules": { "users": { "$uid": { ".read": "auth != null && auth.uid === $uid", ".write": "auth != null && auth.uid === $uid" } } } }
  6. Return here, paste the config, and click Connect.
