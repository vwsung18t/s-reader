Connect Firebase to sync across devices.

Paste from Firebase console → Project settings → Your apps. Must include databaseURL.
Step-by-step setup
  1. Go to console.firebase.google.com → Add project.
  2. Add a Web App → copy the config object.
  3. Authentication → Sign-in method → Google → Enable.
  4. Realtime Database → Create database → Start in test mode.
  5. Rules tab → paste:
    { "rules": { "users": { "$uid": { ".read": "auth != null && auth.uid === $uid", ".write": "auth != null && auth.uid === $uid" } } } }
  6. Return here, paste config, click Connect.
